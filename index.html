About me:

Hello! I'm a master's student studying statistics at UW-Madison and a data science intern at Enveda Biosciences. In my free time, I like to make graphs (especially about the popular reality TV show Survivor). Here, you can find the results of my tinkering. All code used for these graphs is on my GitHub.

Graphs and visualizations:

Survivor:

  • t-SNE with Survivor contestants: Using scikit-learn and D3.js to explore similarities between contestants.
  • No season is an island: An interactive network graph showing how contestants have returned across Survivor seasons.

Other:

  • Grapes: Experimenting with interactive circle-packing in D3.js.
  • Head to head: ggplot2 v. r2d3: Using the r2d3 interface between R and D3.js to recreate a scatterplot from ggplot2.
  • Data Viz with Angular & D3.js: Using the OpenWeather API to make a scatterplot of temperature vs. % humidity for all US Capital Cities.
